# FSD Lab 2 - Learning Management System

Requirements implemented:

1. Navigation Bar
   - Used <nav> tag in index.html
   - Styled using Flexbox in CSS

2. Hero Section
   - Used section with class hero
   - Gives introduction to LMS project

3. Flexbox Layout
   - Used in student dashboard section
   - CSS: display:flex, flex-direction:row

4. Grid Layout
   - Used for course cards and HTML5 API cards
   - CSS: display:grid, grid-template-columns

5. HTML5 APIs
   - Local Storage API: saves student name in browser
   - Geolocation API: displays latitude and longitude
   - Small JavaScript is used because API actions require JavaScript

6. Modal
   - Modal is created using HTML and CSS only
   - Uses CSS :target selector
